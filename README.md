### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 忘记密码
4. 首页
5. 作品信息
6. 公告信息
7. 个人中心
8. 我的收藏
9. 后台管理（作品信息、私聊信息）


**管理员功能：**
1. 登录、忘记密码
2. 个人中心(修改密码、个人信息)
3. 用户管理
4. 作品分类管理
5. 作品信息管理
6. 私聊信息管理
7. 系统管理（关于我们、系统简介、轮播图管理、公告信息）
